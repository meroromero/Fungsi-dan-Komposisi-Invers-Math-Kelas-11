1. Komposisi Fungsi (f o g)(x)

Penggabungan dua fungsi sehingga menghasilkan fungsi baru. Nilai g(x) dimasukkan ke dalam f(x).

(f o g)(x) = f(g(x))

Contoh: f(x) = 2x, g(x) = x + 3

Langkah 1: Tulis f(g(x))
Langkah 2: Masukkan (x + 3) ke dalam x di f(x)
Hasil: 2(x + 3) = 2x + 6

2. Fungsi Invers f-1(x)

Invers adalah fungsi kebalikan. Mencari nilai asal dari sebuah hasil pemetaan.

y = f(x) âž” x = f-1(y)

Trik Cepat Linear:

f(x) = ax + b âž” f-1(x) = (x - b) / a
